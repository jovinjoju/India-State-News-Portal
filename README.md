# India State News Portal
Created by **Jovin Joju**.

A simple, fast, and lightweight web application that fetches real-time news headlines for all Indian States and Union Territories.

### Technologies Used
- HTML5
- CSS3
- JavaScript (Async/Await)
- Google News RSS API

### Features
- Real-time "Top News" updates.
- Fully responsive dark-themed interface.
- 100% Client-side (no server-side API keys required).
